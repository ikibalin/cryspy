Welcome to documentation page. 

Type in console (or terminal):

- **to install:**

{% highlight bash %}
python -m pip install cryspy cryspy_editor

# additional libraries
python -m pip install pyqt5 pyqtgraph scipy matplotlib numpy
{% endhighlight %}

- **to run:**

{% highlight bash %}
python -m cryspy_editor
{% endhighlight %}


- **to upgrade:**

{% highlight bash %}
python -m pip install --upgrade cryspy cryspy_editor
{% endhighlight %}

See examples on [GitHub][cryspy-examples]

[cryspy-examples]: https://github.com/ikibalin/cryspy/tree/master/examples
