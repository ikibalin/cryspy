# Examples

Refinement of various data types is presented:

- Refinement of unpolarized neutron diffraction data on 
  - powder (1D data): Fe3O4, Dy3Al5O12, PbSO4.

- Refinement of polarized neutron diffraction data on 
  - single crystal: Ho2Ti2O7, TbMnO3.
  - powder (1D data): Ho2Ti2O7, Fe3O4.
  - powder (2D data): Ho2Ti2O7, Fe3O4, Dy3Ga5O12.
  - textured sample (2D data): Dy3Al5O12, SMM.

- Reconstruction of spin density by two channel MEM on 
  - single crystal: YTiO3.

- Neutron TOF 1d data refinement: CeCuAl.

- Magnetic space groups for powder data: URu0.96Rh0.04Si2.

Use 'main.rcif' file to open examples in CrysPy Editor.
